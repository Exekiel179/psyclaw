# Abstract

本研究检验 8 周正念减压干预对大学生焦虑的效果。200 名本科生被随机分配至干预组或等待组。ANCOVA 显示干预组后测焦虑显著低于对照组,d = 0.45, 95% CI [0.17, 0.73]。

# 引言

焦虑是大学生最常见的心理健康问题之一。

## 正念与焦虑

已有元分析表明正念干预对焦虑有中等效应。

# 方法

## 被试

经 G*Power 功效分析(α = .05, 功效 = .80, d = .40),每组需 100 人。

### 剔除标准

草率作答按预注册标准筛查(longstring、IRV)。

# 结果

ANCOVA 显示组别主效应显著,F(1, 197) = 10.24, p = .002, η² = .049。

# 讨论

结果支持正念干预的有效性。
