# 样本量 / 功效分析

给研究定样本量时按这个流程走，别拍脑袋、别只报一个点估计。

## 步骤

1. **确定检验类型**：主分析是独立样本 t（两组均值）、单因素 ANOVA（多组）、还是相关（r）？有多个假设时，按最需要功效的那个定 N。
2. **确定效应量来源**（最关键，别乱填）：
   - 优先：前人同类研究的元分析 / 文献效应量；
   - 其次：预试点数据估计；
   - 或：你能接受的**最小有意义效应**（SESOI），而不是"期望"的效应；
   - 交代来源，别拿 Cohen 的"中等"当默认值。
3. **定 α 与 power**：α = .05（双侧）；power = .80（探索）或 .90 / .95（确证 / 高代价研究）。
4. **算 N —— 统计外移，不手算**：生成委托 statsmodels 的脚本来解 N。
   在对话里说「帮我算样本量」，或用 `psyclaw.psych.power_script.generate_power_script(test, effect, alpha, power, groups)` 生成脚本后 `python` 跑（先 `pip install statsmodels`）。
5. **敏感性分析**：报效应量高估 / 低估时的 N（脚本已含），说明 N 对效应量假设有多敏感。
6. **收尾交代**：脱落率上浮（如 ×1.15）、多重比较对 α 的影响、分层 / 嵌套设计的设计效应。

## 红线

- 不做事后功效（post-hoc power 用已得数据倒推，无意义）。
- 效应量要有据，不来自"希望"。
- 统计计算一律外移（statsmodels / scipy），psyclaw 不内联算。
